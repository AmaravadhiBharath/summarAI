## EPISODE 5: "The Technical Nightmare We Solved"
**Duration:** 90 seconds

### SCRIPT

### CHALLENGE 1: MANIFEST V3

**VISUAL:** Chrome Web Store rejection notice

**VOICEOVER:**

*"Challenge number one: Manifest V3."*

**VISUAL:** Code comparison:

**Old way (Manifest V2):**
```javascript
// Easy, but insecure
chrome.tabs.executeScript({
  code: 'document.body.innerHTML'
});
```

**New way (Manifest V3):**
```javascript
// Secure, but complex
chrome.scripting.executeScript({
  target: { tabId: tab.id },
  func: () => {
    // No remote code allowed
    // Everything must be bundled
    // Service workers only
  }
});
```

**VOICEOVER:**

*"Google forced everyone to Manifest V3. No remote code. No eval. Everything sandboxed."*

*"Most extensions broke. We had to rebuild from scratch."*

### CHALLENGE 2: SECURITY

**VISUAL:** Hacker icon, API key being stolen

**VOICEOVER:**

*"Challenge number two: API key security."*

**VISUAL:** Diagram showing:

**Bad approach:**
```

(API key exposed in client code)
‚ùå Hackers steal your key
‚ùå Drain your bank account

**Good approach:**
```
